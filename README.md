# Student Attendance Management System

## 📌 Project Overview
The Student Attendance Management System is a streamlined application designed to track, manage, and report student attendance records efficiently. 

## ✨ Features
- Record daily attendance for students across various B.Tech courses.
- Manage and track student records including their unique ID, Name, Course, and Attendance Status.
- Generate and parse attendance reports in CSV format.
